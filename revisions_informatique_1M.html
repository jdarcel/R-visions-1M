Exercices encore plus exigeants, avec plus d’autonomie, de raisonnement et de combinaison entre les notions.

Exercice 1 – Polygone conditionnel

Turtle Défi
  • Ecrivez un programme qui demande un nombre n .
  • Si n < 3 , le programme affiche impossible .
  • Sinon, il dessine un polygone régulier à n côtés de longueur 40.
Afficher un corrigé / piste 
from turtle import *
n = int(input("Nombre de côtés : "))
if n < 3:
    print("impossible")
else:
    for i in range(n):
        forward(40)
        left(360/n)
done()

Exercice 2 – Spirale conditionnelle

Turtle Défi
  • Ecrivez un programme Turtle de 20 étapes.

  • Si i est pair, la couleur doit être rouge, sinon bleue.
Afficher un corrigé / piste

for i in range(1, 21):

Exercice 5 – Jeu de devinette amélioré

Random Défi
  • Ecrivez un programme avec un nombre aléatoire entre 1 et 50.
  • L’utilisateur doit deviner ce nombre.
  • Le programme doit indiquer plus grand ou plus petit jusqu’à ce que la bonne réponse soit trouvée. Utilisez la fonction randint du module random.
Afficher un corrigé / piste 
from random import *
secret=randint(1, 50)
nb=int(input("Devinez : "))
while nb!=secret:
    if nb<secret:
        print("plus grand")
    else:
        print("plus petit")
    nb=int(input("Devinez : "))
print("Bravo")
